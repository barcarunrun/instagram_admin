# リリースノート

## バージョン: 1.0.0

**リリース日**: YYYY-MM-DD

### 新機能

- **ユーザー登録**: ユーザーが新しいアカウントを作成できるようになりました
- **投稿機能**: ユーザーが写真とテキストで投稿できるようになりました
- **フォロー機能**: ユーザーが他のユーザーをフォローできるようになりました

### 改善

- ログインプロセスの最適化
- ページロード時間を30%削減
- UI/UXの改善

### バグ修正

- メール送信の失敗を修正
- プロフィール画像のアップロードエラーを修正
- 検索機能の不具合を修正

### 已知の問題

- 大容量ファイルのアップロードが遅い場合がある
- 一部のブラウザでアニメーションが正しく表示されない

### マイグレーション

- データベーススキーマが変更されました
- 既存ユーザーデータの移行が自動的に行われます

### セキュリティ

- JWT トークンの有効期限を短縮
- パスワード保存方式を強化
- XSS対策を実装

## アップグレード手順

1. 新しいバージョンをダウンロード
2. 既存データをバックアップ
3. 新しいバージョンをデプロイ
4. データベースマイグレーション実行
5. サービス再起動
