# マイグレーション計画

## データベースマイグレーション

### Version 1.0.0 への移行

**マイグレーション内容**:

1. `users` テーブルに `bio` カラムを追加
2. `posts` テーブルに `likes_count` カラムを追加
3. 新しい `comments` テーブルを作成
4. 既存ユーザーにデフォルト値を設定

**マイグレーション時間**: 約 5 分

**バックアップ**: マイグレーション前に自動的に実行

## マイグレーション実行スクリプト

```bash
# マイグレーション確認
npm run migrate:check

# マイグレーション実行
npm run migrate:up

# マイグレーションロールバック
npm run migrate:down
```

- **ダウンタイム**: 5 分程度
- **データ損失リスク**: なし (バックアップあり)
- **パフォーマンス影響**: なし
