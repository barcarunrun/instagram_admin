# 画面一覧

## 画面リスト

| ID | 画面名 | 説明 | 関連ストーリー |
|--|--|--|--|
| S1 | ログイン / アカウント連携画面 | Facebook OAuthとInstagramビジネスアカウント連携を行う | ストーリー1 |
| S2 | ダッシュボード画面 | 投稿実行率、週次投稿本数、失敗件数を確認する | ストーリー6 |
| S3 | コンテンツ一覧画面 | 下書き・予約済み・要対応の投稿を検索/絞り込みする | ストーリー2, 5 |
| S4 | コンテンツ編集画面 | 投稿種別ごとにメディアとテキストを作成/編集する | ストーリー2, 3 |
| S5 | 予約設定モーダル | 投稿日時、タイムゾーン、公開先を指定して予約する | ストーリー4 |
| S6 | 投稿カレンダー画面 | 日/週/月で予約と実績を可視化する | ストーリー4, 6 |
| S7 | 投稿実行ログ画面 | 投稿結果、再試行履歴、エラー詳細を確認して再実行する | ストーリー5 |
| S8 | 通知 / 監査ログ設定画面 | 通知チャネルと監査ログ閲覧条件を設定する | ストーリー5, 6 |

## 画面関係図

```
S1 ログイン / 連携
  -> S2 ダッシュボード
	  -> S3 コンテンツ一覧
		  -> S4 コンテンツ編集
			  -> S5 予約設定
				  -> S6 投稿カレンダー
	  -> S7 投稿実行ログ
	  -> S8 通知 / 監査ログ設定

主要導線:
- 新規投稿: S2 -> S3 -> S4 -> S5 -> S6
- 失敗復旧: S2 -> S7 -> (再実行) -> S6
- 連携更新: S2 -> S1
```
