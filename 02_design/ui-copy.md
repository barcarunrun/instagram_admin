# UI コピー

## ボタンラベル

| ボタン | テキスト | 代替案 |
|--|--|--|
| 連携開始 | "Facebookで連携する" | "Instagramを接続する" |
| 下書き保存 | "下書きを保存" | "保存" |
| 予約登録 | "この日時で予約する" | "予約を確定" |
| 再実行 | "再実行する" | "もう一度投稿する" |
| テスト通知 | "テスト通知を送信" | "通知を確認する" |
| キャンセル | "キャンセル" | "閉じる" |

## エラーメッセージ

| エラー | メッセージ | 対応 |
|--|--|--|
| 必須項目未入力 | "未入力の必須項目があります。入力してから再度お試しください。" | 未入力項目へ自動スクロール |
| 過去日時予約 | "公開日時は現在より後の時刻を指定してください。" | 日時入力欄にフォーカス |
| メディア形式不正 | "この投稿種別では利用できないメディア形式です。" | 対応形式ガイドを表示 |
| トークン期限切れ | "アカウント連携の有効期限が切れています。再連携してください。" | 連携画面への導線表示 |
| レート制限 | "混雑のため処理を待機しています。自動で再試行します。" | 再試行状況を表示 |

## 通知メッセージ

| 種別 | メッセージ | 表示条件 |
|--|--|--|
| 成功 | "予約を登録しました。指定時刻に自動投稿されます。" | 予約登録成功時 |
| 成功 | "投稿が完了しました。" | 投稿ジョブ成功時 |
| 警告 | "この投稿は要対応です。内容を修正して再実行してください。" | 恒久エラー発生時 |
| 情報 | "自動再試行中です（{retryCount}/{maxRetry}）。" | 一時エラーで再試行時 |
| 情報 | "投稿実行率が目標を下回る見込みです。" | KPI閾値未達見込み時 |

## プレースホルダーテキスト

| フィールド | プレースホルダー |
|--|--|
| 投稿タイトル | "例: 新作シャツ_初夏コーデ_2026W25" |
| キャプション | "商品の魅力や着用シーンを具体的に書いてください" |
| ハッシュタグ | "例: #メンズファッション #夏コーデ #新作" |
| ログ検索 | "投稿ID、エラーコード、キーワードで検索" |
