# リグレッションテスト

## リグレッションテストスイート

リグレッションテストは、変更がシステムの既存機能に悪影響を与えていないことを確認するテストです。

### テストチェックリスト

- [ ] ユーザー登録機能
- [ ] ログイン機能
- [ ] ログアウト機能
- [ ] ユーザープロフィール表示
- [ ] 投稿作成機能
- [ ] 投稿表示機能
- [ ] 投稿削除機能
- [ ] 画像アップロード付き下書き保存
- [ ] 動画メディア付き下書き保存
- [ ] 不正メディアのエラー表示
- [ ] 下書き複製
- [ ] 下書き更新履歴の保存
- [ ] 未来時刻での予約作成
- [ ] 過去日時予約のバリデーションエラー
- [ ] 重複予約の拒否
- [ ] 予約更新
- [ ] 予約取消
- [ ] ダッシュボードで KPI 表示
- [ ] ダッシュボードで失敗投稿と未実行一覧表示
- [ ] ダッシュボードで KPI 未達アラート表示
- [ ] 投稿カレンダーで Day / Week / Month 表示
- [ ] フォロー機能
- [ ] 検索機能

## テスト実行手順

```bash
# リグレッションテストの実行
npm run test:regression

# または
yarn test:regression
```

## テスト実行スケジュール

- **毎回のリリース前**: 必須
- **大規模な変更後**: 必須
- **バグ修正後**: 推奨
- **定期的**: 週1回
