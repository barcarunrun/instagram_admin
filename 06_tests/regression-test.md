# リグレッションテスト

## リグレッションテストスイート

リグレッションテストは、変更がシステムの既存機能に悪影響を与えていないことを確認するテストです。

### テストチェックリスト

- [ ] ユーザー登録機能
- [ ] ログイン機能
- [ ] ログアウト機能
- [ ] ユーザープロフィール表示
- [ ] 投稿作成機能
- [ ] 投稿表示機能
- [ ] 投稿削除機能
- [ ] フォロー機能
- [ ] 検索機能

## テスト実行手順

```bash
# リグレッションテストの実行
npm run test:regression

# または
yarn test:regression
```

## テスト実行スケジュール

- **毎回のリリース前**: 必須
- **大規模な変更後**: 必須
- **バグ修正後**: 推奨
- **定期的**: 週1回
